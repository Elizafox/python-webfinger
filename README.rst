================
python-webfinger
================

Usage
=====

Example::

	from pywebfinger import finger
	
	res = finger('user@host.com')

Dependencies
============

* `python-xrd <http://github.com/jcarbaugh/python-xrd>`_
* iso8601 (required by python-xrd)
